# Python

Python development container with [uv](https://docs.astral.sh/uv/) pre-installed.

## Usage

```json
{
  "image": "ghcr.io/tbhb/devcontainer-python:3.14"
}
```

## Features

- Python 3.10 - 3.14
- uv package manager
- Zsh shell with common-utils
- Ruff extension pre-configured
