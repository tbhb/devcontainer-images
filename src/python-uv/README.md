# Python with uv

Python development container with [uv](https://docs.astral.sh/uv/) pre-installed.

## Usage

```json
{
  "image": "ghcr.io/tbhb/python-uv:latest"
}
```

## Features

- Python 3.13 (default) or 3.12
- uv package manager
- Ruff extension pre-configured

## Variants

| Tag | Python Version | Base |
|-----|---------------|------|
| `3.13-bookworm` | 3.13 | Debian Bookworm |
| `3.12-bookworm` | 3.12 | Debian Bookworm |
| `latest` | 3.13 | Debian Bookworm |
